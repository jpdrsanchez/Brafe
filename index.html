Brafé

Projeto final do curso "CSS avançado posicionamento" neste projeto foram criados 3 layouts diferentes. Cada um dos layouts foi codificado utilizando 4 maneiras diferentes. A primeira da forma a mais pura possível, a segunda com um sistema de grid pré definido, a terceira com o Flexbox e por último usando o Bootstrap.
